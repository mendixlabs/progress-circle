# ProgressCircle
Displays a circular progress bar.

## Features
* Shows the percentage (from 0 to 100) based on the progress value and the maximum progress value
* Animates on load

## Dependencies
Mendix 6.

## Demo project
http://progresscircle.mxapps.io
